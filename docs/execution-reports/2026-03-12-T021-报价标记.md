## 目标

在contacts表中标记已收到销售报价的客户（has_quote + first_quote_at），供看板漏斗"已报价"层使用。

## 执行内容

### 1. Schema变更

contacts表新增两个字段：
- `has_quote` BOOLEAN DEFAULT false
- `first_quote_at` TIMESTAMPTZ

SQL文件：`sql/t021_quote_fields.sql`

### 2. 报价识别脚本

脚本：`scripts/mark_quotes.py`

**扫描条件**：
- sender_type = 'sales'（只看销售发出的消息）
- is_system_msg = false
- msg_type = 1（纯文本）
- 排除群聊（room_id含@chatroom）

**匹配规则（任一命中即算报价）**：
1. 含 ¥ + 数字
2. 含 报价/优惠价/折后/特价/活动价
3. 含 数字+元
4. 含 数字+万
5. 含4-6位连续数字 + 产品关键词（价/沙发/茶几/床/柜）共现

**排除规则**：
- 含11位连续数字（手机号）
- 含 电话/微信号/手机 关键词
